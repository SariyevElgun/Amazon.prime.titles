# 📊 Amazon Prime Video Dashboard (Power BI)

I am excited to share my latest Data Analytics project! This repository features a comprehensive Amazon Prime Video Dashboard developed using Power BI.

## 🚀 Project Overview
This project explores a vast dataset of over **9,600 titles**, covering movies and TV shows from 1920 to 2021. The goal was to provide deep insights into content distribution, audience ratings, and historical production trends.

## 🔍 Key Insights
* **Content Mix:** Movies dominate the platform, making up over **80%** of the total library.
* **Demographics:** The "13+" rating category holds the highest number of shows, followed closely by "16+".
* **Top Genres:** Drama and Comedy remain the most popular genres globally.
* **Growth Trend:** A massive surge in content production is visible starting from the early 2000s.
